Análisis de Sentimientos con Modelos Pre-entrenados (VADER vs RoBERTa)

## Este proyecto tiene como objetivo comparar el desempeño de dos enfoques distintos de análisis de sentimientos:  
- **VADER (NLTK) — un modelo léxico basado en reglas.  
- **RoBERTa (HuggingFace) — un modelo de lenguaje preentrenado basado en *transformers*.

## Objetivo del proyecto

Analizar y comparar cómo dos modelos distintos interpretan el sentimiento de reseñas de productos de Amazon, evaluando su precisión, sesgos y diferencias en confianza.

El dataset utilizado es **[Amazon Fine Food Reviews (Kaggle)](https://www.kaggle.com/datasets/snap/amazon-fine-food-reviews)**, que contiene más de 500,000 reseñas de clientes.
